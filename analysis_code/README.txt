Instruction to create the video on Swarm Q Learning

1. Open the makeVideo.m file in MATLAB.

2. Run the command: makeVideo(NumAgent,RL_method,episode)

	E.g: makeVideo(16,'SQL-SIE',2500) # to create a video of SQL-SIE method on environment including 16 agents at Episode 2500.

3. When the file selection dialog box pops up, select the right data csv file named "step_swarmxxx.csv".

4. The video will be saved in the same folder. Make sure to check it out!
